Gesture-Based App Launcher

Overview

The Gesture-Based App Launcher is an innovative project that combines computer vision, voice recognition, and biometric authentication to create a secure and hands-free way of launching applications. It demonstrates how human-computer interaction (HCI) can move beyond traditional input devices like keyboards and mice.

Why it matters

This project highlights the future of interaction with devices by using natural input methods such as gestures and voice. It also ensures security through facial recognition and voice authentication, making it suitable for both personal use and enterprise environments.

Features

  • Gesture Recognition: Uses a camera to detect and interpret hand gestures for app selection and launching.
  • Voice Commands: Provides a hands-free experience by allowing apps to be launched using spoken commands.
  • Multi-layered Security: Ensures safe access using both facial recognition and voice authentication.
  • Graphical Interface: Built with Tkinter for a user-friendly and responsive GUI.
  • Gesture Click Mechanism: Users can “click” on applications using hand gestures detected by the system.
  • Expandable Design: Integrated with a web signup/login system and adaptable for more advanced controls.

Tech Stack

  • Python
  • OpenCV
  • Tkinter
  • SpeechRecognition
  • Flask (for web integration)
  • Machine Learning for face and voice authentication
