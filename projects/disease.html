Disease Prediction & Analysis

Overview

The Disease Prediction & Analysis system is a Flask-based web application that uses machine learning models to predict possible diseases based on user symptoms. It combines multiple classifiers and an ensemble approach to achieve accurate and reliable predictions, while also providing users with essential health-related information.

Why it matters

Early detection of diseases can play a vital role in timely treatment and better healthcare outcomes. This project provides an accessible tool that allows users to get preliminary health insights before consulting a medical professional, making it a useful application in AI for healthcare.

Features

  • User Authentication: Login/Signup system with secure session handling.
  • Multiple ML Models: Predictions from Decision Tree, Random Forest, KNN, Naïve Bayes, SVM, and a Voting Classifier.
  • Performance Metrics: Displays accuracy, precision, recall, and F1-score for each model.
  • Data Visualization: Graphical analysis of symptom–disease datasets using charts and plots.
  • Disease Information: Provides details on precautions, treatment methods, severity levels, and fatality scores.
  • User-Friendly Interface: Simple design to ensure accessibility for non-technical users.

Tech Stack

  • Python
  • Flask
  • scikit-learn
  • Pandas, NumPy
  • Matplotlib
  • Chart.js
  • SQLite
