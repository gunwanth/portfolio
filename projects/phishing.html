Phishing Email Detection App

Overview

The Phishing Email Detection App is a Streamlit application integrated with Gmail that scans inbox emails and detects phishing threats using natural language processing (NLP) techniques. It provides a risk score and highlights suspicious indicators for better email security.

Why it matters

With phishing being one of the most common cyberattacks, this project demonstrates practical cybersecurity applications by combining AI-powered email analysis with real-world Gmail integration. It helps users identify and avoid malicious emails.

Features

  • Secure Gmail authentication using OAuth2.
  • Scans inbox emails for suspicious content and malicious links.
  • Classifies risk levels: Low, Medium, or High.
  • Highlights phishing indicators such as blacklisted URLs or suspicious keywords.
  • Supports multiple users with login/signup functionality.
  • Designed for real-world deployment with alerts and notifications.

Tech Stack

  • Python
  • Streamlit
  • Google OAuth2
  • Gmail API
  • NLP (Natural Language Processing)
  • BeautifulSoup
